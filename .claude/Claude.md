## SEO Principles

### Structure
- Use semantic HTML (`header`, `main`, `section`, `article`, `nav`, `footer`)
- Proper heading hierarchy (`h1` â†’ `h2` â†’ `h3`)
- Only ONE `h1` per page

### Metadata
- Always include:
  - `<title>` (clear + keyword-focused)
  - `<meta name="description">`
- Use meaningful, human-readable URLs

### Content
- Write content for humans first, SEO second
- Use relevant keywords naturally
- Avoid keyword stuffing

### Performance (SEO factor)
- Optimize images (lazy loading, modern formats)
- Minimize JS/CSS
- Ensure fast load times

### Accessibility (SEO impact)
- Use `alt` text for images
- Proper labels for inputs
- Logical structure for screen readers
